# EcoNexus - Smart Waste Mapping Solution

### 🎓 Academic Project
This is a University Extension Project developed to solve real-world community issues through technology and geographic intelligence. **EcoNexus** acts as a digital bridge between citizens and proper disposal points for hazardous waste.

---

## 🚀 The Problem
In my local community, a significant amount of **special waste** (electronics, batteries, and cooking oil) is disposed of in regular trash. This happens not due to a lack of will, but due to a **"gap of information"**: residents don't know exactly *where* or *when* they can drop off these materials.

## 💡 The Solution (MVP)
**EcoNexus** is a Minimum Viable Product (MVP) focused on "Geographic Intelligence". It simplifies the recycling journey by:
* **Real-time Mapping:** Locating the nearest Collection Points (PEVs).
* **Smart Filtering:** Categorizing points by waste type (E-waste, Oil, or Batteries).
* **Logistical Efficiency:** Providing operating hours and contact info to avoid lost trips.

---

## 🛠️ Tech Stack & Methodology
* **Frontend:** HTML5 & CSS3 (Mobile-first and responsive design).
* **Logic:** Vanilla JavaScript for search simulation and UI interactions.
* **Management:** **Scrum Framework** adapted for individual development (Sprints & Kanban).
* **Design:** Prototyped using **Figma** principles for high usability.

---

## 📈 Key Features (MVP Scope)
1. **CEP/Neighborhood Search:** Quick access to local data.
2. **Category Selection:** Visual icons for intuitive navigation.
3. **Status Feedback:** Instant confirmation of active collection points.
